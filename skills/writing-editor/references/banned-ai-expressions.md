# Banned AI-Template Expressions

Use this as a detection list, not a mechanical replacement list. If an expression carries real information in context, rewrite it instead of deleting it blindly.

## Empty Openings

- 在当今快速发展的时代
- 随着技术的不断发展
- 近年来，随着……
- 在数字化转型的大背景下
- 面对日益复杂的业务需求
- 众所周知
- 不得不提的是

## Template Transitions

- 值得注意的是
- 不难发现
- 可以看出
- 从某种意义上说
- 在这个过程中
- 与此同时
- 进一步来说
- 毫无疑问

## Empty Summaries

- 具有重要意义
- 提供了新的思路
- 带来了更多可能性
- 是一个值得深入思考的问题
- 为后续发展奠定了基础
- 未来仍有很大的想象空间

## Mechanical Structure

- 首先，其次，最后
- 一方面，另一方面
- 不仅如此，更重要的是
- 综上所述
- 总而言之
- 本文将从以下几个方面展开

## Rewrite Rule

When a sentence only frames the article and adds no information, delete it.

When a sentence hides a judgment behind abstract wording, replace it with the actual judgment, tradeoff, or project constraint.
